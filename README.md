AI-Powered Study Buddy

Project Title:

AI-Powered Study Buddy using Python and Groq API

Project Description:

The AI-Powered Study Buddy is a web-based chatbot application developed using Python and

Streamlit. The system assists students in understanding academic concepts by providing simplified

explanations, summarizing study materials, and generating quizzes for self-assessment. The

application integrates a Large Language Model through the Groq API to deliver intelligent

responses and also supports PDF document uploads for study material analysis.

Objectives:

• Develop an AI-based study assistant.

• Simplify complex academic topics.

• Generate automatic summaries of study notes.

• Create quizzes for knowledge evaluation.

• Deploy as a web application accessible via browser.

Technologies Used:

• Python 3.12

• Streamlit Framework

• Groq API

• OpenAI Library

• python-dotenv

• PyPDF2

System Architecture:

• User enters topic or uploads PDF file.

• System processes the input.

• Groq API generates intelligent response.

• Output displayed via Streamlit interface.

Expected Outcome:

• Students understand topics in simple language.

• Users summarize long notes instantly.

• System generates quizzes for practice.

• Application accessible online via public URL.

Future Enhancements:

• Voice input integration.

• User authentication system.

• Quiz performance analytics.

• Dark mode interface.
